Build a basic employee payroll dashboard

Use Case:

You are building a secure Payroll lookup system for HR. HR would like a system that shows all associate names and their Id. Upon clicking the ID, show the employee details on screen.

Requirements:

  1. Add Bootstrap or Material Design to the project for some component styling.
  2. Display all associates and their corresponding ID ('All Employees' API listed below)
  3. Order employees by name
  4. Clicking on a row, will result in displaying that associates details on the screen (Must use Single Employee Detail API below for details)
  5. Add a search to filter the employees by name.
  6. Add necessary unit test for the components you built.
  7. Implement a simple login, with a hard-coded username and password (hr_admin:12345). Base64 encode the username:password and add it to sessionStorage:authToken
  8. Inject the authToken into all API requests with like so
    • Authorization: Bearer authToken
  9. Update the app so the login is required before rendering the employee data

Use the following REST endpoint

  • All Employees (http://dummy.restapiexample.com/api/v1/employees)
  • fallback API endpoint (https://hub.dummyapis.com/employee)
